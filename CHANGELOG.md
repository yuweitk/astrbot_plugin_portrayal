# 更新日志

## v1.1.5

新功能：

    添加基于 JSON 的消息缓存存储，在多次运行之间保留每个用户的消息和群组游标。
    引入一个新的内置提示命令“找对象”，根据聊天记录推荐合适的对象类型。

错误修复：

    确保过期的群组缓存被完全清理并持久化到磁盘，避免重复使用过期消息。

改进：

    使用按群组划分的 asyncio 锁保护并发的群组历史扫描，以安全共享扫描进度。
    优化消息查询的元数据和文档字符串，使行为和意图更加清晰。
    将 portrayal 和 clone 命令的输出简化为纯文本响应，并移除对图像渲染的依赖。

## v1.1.4

- 为每个提示词条目加上“仅管理员可用”的选项，增强对触发指令权限的控制
